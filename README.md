Sales analysis and forecasting system.
My goal is to create a Python application that analyzes sales data, stores the results in a SQL database, and generates reports.
Scope of work:
Pandas / NumPy: Sales aggregations (e.g., daily revenue, margin, bestsellers).
Vector operations: Calculating trends, moving averages, and variances.
SQLAlchemy: Loading and saving data to the database (SQLite / PostgreSQL).
ORM for table definitions (Sales, Products, Customers).
Pydantic: Input data validation (correct date format, price, product ID).
Additional features: Report export to CSV/Excel.
Simple CLI or web interface (FastAPI + Pydantic for endpoints).
